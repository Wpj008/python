# Exercices en Python

Ce dépôt contient une série d’exercices écrits en langage Python visant à pratiquer les concepts fondamentaux de la programmation ainsi que la programmation orientée objet (POO).

## Contenu
Les exercices abordent les notions suivantes :
- Déclaration et manipulation des variables
- Opérations et calculs
- Structures conditionnelles (if, elif, else)
- Boucles (for, while)
- Fonctions
- Programmation Orientée Objet (classes, objets, méthodes, constructeurs, destructeurs)

## Objectif
Acquérir une bonne compréhension du langage Python à travers des exercices progressifs, du niveau débutant aux notions intermédiaires.
